pass_krab


The map pass_krab created by moose for the Team Fortress 2 4v4 PASS Time community. The map was the first initially only made for the EU branch of 4v4 PASS Time. It is designed to be used with no-lock; you cannot lock onto others for passes. Some versions like _m1 & _magnet also have magnet passing enabled. From v2 onwards (versioning protocols fear moose), the base version has magnet passing, the _nolock version has no lock passing.
